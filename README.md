# Agriculture Yield Prediction and Analysis

## Overview

This project analyzes an agricultural dataset containing information about rainfall, temperature, fertilizer usage, irrigation hours, soil characteristics, and crop types.

The objective is to perform:

- Data Understanding
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Feature Engineering
- Linear Regression Modeling

---

## Dataset Features

| Feature | Description |
|----------|------------|
| rainfall_mm | Annual rainfall in millimeters |
| temperature_c | Average temperature in Celsius |
| fertilizer_kg | Fertilizer used in kilograms |
| irrigation_hours | Irrigation hours |
| soil_ph | Soil pH level |
| crop_type | Type of crop |
| soil_type | Type of soil |
| yield_ton_per_hectare | Crop yield per hectare |

---

## Key Findings

- Rainfall has a strong positive impact on crop yield.
- Irrigation contributes significantly to yield improvement.
- Rice shows the highest average yield among crop categories.
- Loamy soil produces higher yields compared to other soil types.
